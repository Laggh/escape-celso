# Escape celso

Projeto para a feira de ciências da ETEC Jõao Belarmino - 2025

## Como rodar
- Baixe o projeto
- Mude o fim do arquivo de `.zip` para `.love`
- Execute o arquivo `.love` com o [LOVE2D](https://love2d.org/) instalado
- Se divirta
